DOWNLOADING DATA VIA FTP
You may download data via your FTP program of choice. For example, using a 
command-line FTP program, you can download all of the Belcher Islands Sea Ice 
Observations files using the following commands:

%ftp sidads.colorado.edu 
     (login as anonymous, and use your e-mail address as password)
ftp>cd /pub/DATASETS/eloka002_belcher_seaice_v01/
ftp>get ELOKA002_Distribution_Package.zip

In most Web browsers, you can also download the data by going to the address:
ftp://sidads.colorado.edu/pub/DATASETS/eloka002_belcher_seaice_v01/ELOKA002_Dist
ribution_Package.zip

The browser should prompt for where to save the file.

DOCUMENTATION
For documentation and more information about data, please see:

http://eloka-arctic.org/communities/sanikiluaq/index.html

The Web site explains the history of the region, contains participant 
biographies, and summaries of the interviews.  We recommend that you read the 
documentation in detail before working with the data.
